# PageSets — Issue Tracker

**Save and relaunch your Chrome tab groups as named workspaces.**
Stop recreating the same tabs every morning — one click and you're back.

---

## About This Repository

This repository is the **public issue tracker** for the PageSets Chrome extension.
The extension source code is not hosted here.

Use this repo to:
- 🐛 Report bugs
- 💡 Request features or improvements
- 📋 Check whether a known issue has already been filed

For billing questions, license issues, or anything private, please use the [contact form](https://pagesets.co/contact) instead.

---

## Before You File an Issue

1. **Search first** — check [open issues](https://github.com/lumicorelabs/pagesets/issues) to see if it's already been reported
2. **Check the FAQ** — [pagesets.co/faq](https://pagesets.co/faq) covers the most common questions
3. **One issue per report** — please don't bundle multiple bugs or requests into a single issue

---

## Reporting a Bug

[🐛 Open a bug report](https://github.com/lumicorelabs/pagesets/issues/new?labels=bug)

Please include:
- **Chrome version** (Help → About Google Chrome)
- **Operating system** (macOS, Windows, Linux)
- **PageSets version** (shown in the extension popup footer)
- **Steps to reproduce** — what did you do, what did you expect, what happened instead
- **Pro or Free?** — some features are Pro-only

---

## Requesting a Feature

[💡 Open a feature request](https://github.com/lumicorelabs/pagesets/issues/new?labels=enhancement)

A good feature request explains:
- **The problem** you're trying to solve (not just the solution)
- **How you'd use it** — real workflow context helps a lot
- **Why it matters** — who else might benefit
